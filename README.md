# xAI-Classifier-Implementation
Project Overview

This repository contains the implementation of the K-Nearest Neighbors (KNN) algorithm for a supervised machine learning classification task.
KNN is a distance-based algorithm that classifies new data points by examining the labels of the closest training samples.

Features
KNN model implementation
Data preprocessing and training
Prediction on unseen data
Model evaluation
Evaluation Metrics

The model performance is evaluated using:
Accuracy
Precision
Recall
F1-score

Note
This work was originally part of a team project comparing multiple classifiers (KNN, SVM, Random Forest). The code in this repository focuses only on the KNN implementation, which was the component developed for this contribution.
